Unprotect Files
---------------

The following collection of "unprotects" was obtained from [The Programmer's Corner](https://www.pcorner.com/list/UNP/UNPROT-1.ZIP/INFO/).

For example, [LOTUS.TXT](https://github.com/jeffpar/pcjs-disks/blob/master/pcx86/shareware/unprot/disk01/unprot-1/LOTUS.TXT)
describes how to disable the copy protection in [Lotus 1-2-3 Release 1A](/disks/pcx86/apps/lotus/123/1a/#copy-protection-patch) and
[Lotus 1-2-3 Release 1A*](/disks/pcx86/apps/lotus/123/1as/#copy-protection-patch).
